Agilent 4263B LCR Meter Automation

Real-Time Capacitance Measurement for Sensor Characterization

LabVIEW GPIB Agilent 4263B Capacitance Real-Time Plotting

Developed LabVIEW GUI for automated capacitance measurement using Agilent 4263B Precision LCR Meter via GPIB (NI GPIB-USB-HS). Features include configurable test frequency (100 Hz to 100 kHz), measurement averaging (1-256 samples), real-time capacitance vs. time plotting, and automated data logging to Excel/TDMS. Enabled characterization of supercapacitive pressure sensors with 0.1 pF resolution over 5000+ cycles.

SCPI Commands Used:

*RST                    ; Reset to default state
:FUNCtion "CPD"         ; Measure Capacitance (C) and Dissipation (D)
:FREQuency 1000         ; Set test frequency to 1 kHz
:VOLTage 1.0            ; Set test signal level to 1V
:READ?                  ; Trigger measurement and read result

Specifications

  • Instrument: Agilent 4263B
  • Interface: GPIB (IEEE-488)
  • Controller: NI GPIB-USB-HS
  • Capacitance Range: 0.01 pF to 99.9999 mF
  • Basic Accuracy: ±0.1%
  • Sampling Rate: Up to 10 Hz

Applications

  • Paper-based supercapacitive pressure sensor (ACS AMI 2023)
  • 5000-cycle stability testing
  • Pulse waveform monitoring

MARK-10 Test Stand & Force Gauge Automation

Programmable Pressure Profiles for Sensor Calibration

LabVIEW RS-232 MARK-10 Force Gauge Cyclic Testing

Developed LabVIEW automation for MARK-10 ES-20 motorized test stand with M5-50 force gauge via RS-232 serial communication. Features programmable loading/unloading profiles (ramp, hold, cyclic, custom), real-time force vs. time plotting, and automated cyclic testing up to 10,000+ cycles. Enabled comprehensive characterization of piezoresistive and capacitive pressure sensors.

Serial Commands:

?                       ; Request current force reading
Response: +0123.45      ; Force in grams (or N, lb)

U0                      ; Set units to grams
U1                      ; Set units to Newtons
T                       ; Tare the gauge (zero offset)

Specifications

  • Test Stand: MARK-10 ES-20
  • Force Gauge: MARK-10 M5-50
  • Interface: RS-232 Serial
  • Travel Speed: 0.5-50 mm/min
  • Max Load: 50 lbf (250 N)
  • Position Resolution: 0.01 mm

Applications

  • Piezoresistive sensor calibration (19.8 kPa⁻¹)
  • 10,000-cycle durability testing
  • Loading-unloading profiling

MAX30102 Pulse Oximeter Interface

PPG Waveform Acquisition & Heart Rate Monitoring

LabVIEW I2C Arduino MAX30102 PPG

Developed LabVIEW interface for MAX30102 pulse oximeter via Arduino I2C bridge. Features real-time PPG waveform display (red and IR channels), heart rate calculation using FFT-based peak detection, SpO₂ estimation, and data synchronization with pressure sensor signals for cross-validation studies in cardiovascular monitoring.

Arduino Bridge Code (Simplified):

#include "MAX30105.h"
MAX30105 particleSensor;

void setup() {
  Serial.begin(115200);
  particleSensor.begin(Wire, I2C_SPEED_FAST);
}

void loop() {
  if (Serial.available()) {
    char cmd = Serial.read();
    if (cmd == 'R') {
      long ir = particleSensor.getIR();
      long red = particleSensor.getRed();
      Serial.print(red);
      Serial.print(",");
      Serial.println(ir);
    }
  }
}

Specifications

  • Sensor: MAX30102
  • Interface: I2C (via Arduino Uno)
  • LED Wavelengths: 660nm (Red), 880nm (IR)
  • Sampling Rate: 100 Hz
  • Heart Rate Range: 30-200 BPM
  • SpO₂ Range: 70-100%

Applications

  • PPG waveform acquisition
  • Cross-validation with pressure sensors
  • Dual piezoresistive + PPG study (ACS AMI 2024)

Common LabVIEW Architecture

Design Patterns Used:

  • State Machine: Main VI architecture for instrument control (Initialize → Configure → Measure → Log → Plot → Error → Close)
  • Producer-Consumer: For high-speed data acquisition (Producer: data reading, Consumer: logging and plotting)
  • SubVI Modularization: Reusable instrument drivers (e.g., Agilent_4263B_Read.vi, MARK10_Read.vi)
  • Event Structure: For responsive GUI user interactions
