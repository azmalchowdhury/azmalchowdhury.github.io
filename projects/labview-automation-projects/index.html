Automated Test Systems
LabVIEW Integration & Data Acquisition

Multi-Instrument Automation | Real-Time Data | GUI-Based Testing

Overview

Developed LabVIEW-based automation systems to streamline experimental testing and enable reliable, repeatable data generation. These platforms integrate multiple instruments into a unified interface, allowing real-time visualization and direct Excel-based data logging.

The systems were designed to eliminate manual measurements and support scalable testing workflows for sensor characterization.

Key Capabilities

  • Automated testing using custom LabVIEW GUI
  • Real-time visualization of sensor signals
  • Synchronized multi-instrument data acquisition
  • Direct Excel-based data generation
  • Reduced manual error and improved repeatability
---

Multimodal Testing

The system supports combined sensing modes by integrating multiple instruments into a single workflow:

  • Capacitive + Piezoresistive sensing
  • Capacitive + PPG sensing
  • Resistive + PPG sensing

This enables simultaneous acquisition of electrical, mechanical, and physiological signals for deeper device analysis.

---

Impact

This automation framework significantly improved testing efficiency and data reliability. It enabled scalable experimentation and served as a core platform for sensor development, validation, and performance analysis.
